Solution Overview

ResumeAI v2 transforms from a single-purpose resume tailoring tool into an end-to-end AI career acceleration platform. The solution maintains the unique 4-stage Claude pipeline with hallucination validation while adding trust mechanisms (AI confidence scores, batch actions), closing workflow gaps (Chrome extension, job tracker), and completing the application package (discoverable cover letters, template preview). Strategic positioning: 'Claude-powered accuracy at Teal speeds, half the price.'

Current State

V1 provides isolated resume tailoring with 4-stage Claude pipeline (parse, analyze, rewrite, validate) achieving measurable ATS improvements. However, users face trust gaps requiring 20-minute manual verification, workflow friction from manual job description copying and lost tailored versions, and incomplete application packages with only 12% discovering cover letter generation. The Stage 4 hallucination validation moat is invisible to users.

Target State

V2 becomes a trusted career acceleration partner owning the entire workflow from job discovery through interview preparation. Users see AI confidence scores directing review effort to low-confidence bullets only (7-minute review vs. 18 minutes), use Chrome extension for 1-click tailoring from job boards, track applications with linked resume versions, generate matching cover letter PDFs with 50% opt-in, and preview templates before committing to tailoring process.

Key UI Surfaces

  2. Resume upload interface with template preview

  4. Diff review UI with confidence scores and batch actions

  10. Upgrade/paywall modals

Critical User Flows

Core Tailoring Flow (Web)
User lands on homepage → Previews 5 templates in gallery → Selects template → Uploads resume (PDF/DOCX) → Pastes job description → System runs Stage 1-4 pipeline (90 sec) → Diff review UI shows bullets with confidence badges → User applies batch actions ('Accept all high-confidence') or individual edits → Downloads tailored resume PDF → Post-download modal offers cover letter generation → User opts in → Cover letter compiles in 30 sec → Downloads matching PDF
Extension-Assisted Flow
User browses LinkedIn job posting → Clicks 'Tailor with ResumeAI' extension button → Extension extracts JD, authenticates via Supabase → Popup shows user's default resume with 'Tailor Now' CTA → User clicks → Backend runs Stage 1-4 → Extension displays download link → User clicks 'Save to Tracker' → Job auto-added to dashboard with Applied status
Freemium Conversion Flow
New user completes 3 free tailoring sessions → Attempts 4th tailor → System displays upgrade modal: 'You've saved 90 minutes this month—unlock unlimited for $9.99' → User clicks 'Upgrade' → Stripe Checkout opens with Pro/Premium options → User completes payment → Webhook syncs subscription to Supabase → User redirected to dashboard with 'Pro' badge → Unlimited tailoring enabled
Job Tracker Flow
User downloads tailored resume → Checks 'Save to Tracker' → System creates job entry with company/title from Stage 2 analysis → User navigates to Tracker dashboard → Sees Kanban board with Applied column → Drags job card to Interviewing when scheduled → Clicks job card → Views linked resume version → Downloads exact PDF sent to company → Adds notes: 'Follow up on Friday'
Interview Prep Flow (Premium)
Premium user completes tailoring → Navigates to Interview Prep → Selects tailored resume + JD → System generates 10 questions via Haiku → User records/types answer to Q1 → Submits → Sonnet evaluates against STAR framework → Displays score (7/10) with feedback: 'Strong situation setup, but add quantified result' → User revises answer → Re-submits → Score improves to 9/10 → User proceeds to Q2
